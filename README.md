# Touchless Gesture Drawing System

A Python-based touchless drawing application using OpenCV and MediaPipe.

## Features
- Hand gesture drawing
- Color selection
- Eraser mode
- Save drawing
- Touchless interaction

## Technologies Used
- Python
- OpenCV
- MediaPipe
- NumPy

## Controls
- Index Finger → Draw
- Index + Middle Finger → Select Color
- C → Clear Screen
- S → Save Drawing
- Q → Quit
